LES BALISES HTML

Les sélécteurs de type ciblent les éléménts de la page HTML par les balises. Par exemple si nous voulons cibler ts nos titres de niveau <h4> de la page HTML, nous utiliserons le selecteur de type h4.

HTML : <h4>Bonjour à tous </h4>
<h4>Au revoir à tous </h4>

CSS : <h4>{
...}

Les sélecteur de class permettent de cibler un élément en fonction de la valeur de l'attribut class attaché à cet élément. Les selecteurs de class permettent davantage de précision car ils permettent de cibler les éléments d'un groupe ayant une class identique plutôt que toutes les balises HTML de la page. Les sélécteurs de class permettent d'appliquer la même mise en forme en attribuant la même valeur dans l'attribut class à diffférentes balises HTML. En CSS les class sont appelées précédées d'un point. Ici le sélecteur de class ciblera tous élément contenant la class dont la valeur est "zone1", y inclus les élémnts de division et paragraphe.

Les sélecteur d'ID sont encore plus précis que les sélecteurs de class, puisqu'ils ciblent un élément unique. Tout comme les sélecteurs de class utilisent la valeur d'une class présente de le HTML, les sélecteurs d'ID utilisent la valeur de l'attribut id.

:petit_triangle_rouge::petit_triangle_rouge::petit_triangle_rouge: ATTENTION ! Très important : peu importe la balise dans laquelle ils sont insérés, la valeur des attributs ID ne peuvent être utilisés qu'une seule fois dans un fichier HTML. C'est pourquoi les ID sont réservés pour les éléments significatifs.
En CSS les ID sont appelés précédés d'un dièse (ou hash tag). Ici le sélecteur d'ID ciblera uniquement l'élément HTML qui possède un attribut ID dont la valeur est "zone3".
