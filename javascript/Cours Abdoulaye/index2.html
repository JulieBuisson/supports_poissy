La Boucle while

La Boucle while va nous permettre de répéter une serie d'action tant qu'une condition est vraie.

La Boucle do..while

La Boucle for

Elle nous permet de répéter une série d'instruction à partir d'une valeur donnée jusqu'à une certaine valeur seuil.

La fonction

C'est un bloc de code qui va éxécuter une tâche précise.
